A selectively curated list of skills, agents, plugins, hooks, and other amazing tools for enhancing Claude Code workflows. This list takes a selective (rather than exhaustive) curation approach.

**Agent Skills:**
Model-controlled configurations (files, scripts, resources, etc.) that enable Claude Code to perform specialized tasks.

- [Claude Scientific Skills](https://github.com/K-Dense-AI/claude-scientific-skills) — "A set of ready-to-use Agent Skills for research, science, engineering, analysis, finance and writing." — One of the best skills repos on GitHub.

- [AgentSys](https://github.com/avifenesh/agentsys) — Workflow automation system with plugins, agents, and skills. Automates task-to-production, PR management, code cleanup, performance investigation, drift detection, and multi-agent code review.

- [Book Factory](https://github.com/robertguss/claude-skills) — A comprehensive pipeline of Skills that replicates traditional publishing infrastructure for nonfiction book creation.

- [cc-devops-skills](https://github.com/akin-ozer/cc-devops-skills) — Immensely detailed set of skills for DevOps Engineers with validations, generators, shell scripts and CLI tools.

- [Claude Code Agents](https://github.com/undeadlist/claude-code-agents) — Comprehensive E2E development workflow with Claude Code subagent prompts for solo devs. Run multiple auditors in parallel, automate fix cycles.

- [Claude Codex Settings](https://github.com/fcakyon/claude-codex-settings) — Well-organized plugins covering core developer activities with cloud platforms (GitHub, Azure, MongoDB, Tavily, Playwright).

- [Claude Mountaineering Skills](https://github.com/dreamiurg/claude-mountaineering-skills) — Automates mountain route research for North American peaks by aggregating 10+ mountaineering sources.

- [Codex Skill](https://github.com/skills-directory/skill-codex) — Enables users to prompt Codex from Claude Code with inferred parameters.

- [Compound Engineering Plugin](https://github.com/EveryInc/compound-engineering-plugin) — Pragmatic set of well-designed agents, skills, and commands built around turning past mistakes into lessons.

- [Context Engineering Kit](https://github.com/NeoLabHQ/context-engineering-kit) — Hand-crafted collection of advanced context engineering techniques with minimal token footprint.

- [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) — Top-notch resources covering core engineering domains with significant standalone value.

- [Fullstack Dev Skills](https://github.com/jeffallan/claude-skills) — 65 specialized skills covering full-stack development with 9 project workflow commands for Jira/Confluence integration.

- [read-only-postgres](https://github.com/jawwadfirdousi/agent-skills) — Read-only PostgreSQL query skill for Claude Code with strict validation, timeouts, and row limits.

- [Superpowers](https://github.com/obra/superpowers) — Strong bundle of core competencies for software engineering with good SDLC coverage.

- [Trail of Bits Security Skills](https://github.com/trailofbits/skills) — Professional collection of security-focused skills for code auditing and vulnerability detection including CodeQL and Semgrep.

- [TÂCHES Claude Code Resources](https://github.com/glittercowboy/taches-cc-resources) — Well-balanced set of subagents, skills, and commands with focus on meta-skills like skill-auditor and hook creation.

- [Web Assets Generator Skill](https://github.com/alonw0/web-asset-generator) — Generate web assets from Claude Code including favicons, app icons, and social media meta images.

**Workflows & Knowledge Guides:**
Tightly coupled sets of Claude Code-native resources for specific projects.

- [AB Method](https://github.com/ayoubben18/ab-method) — Spec-driven workflow transforming large problems into focused, incremental missions.

- [Agentic Workflow Patterns](https://github.com/ThibautMelen/agentic-workflow-patterns) — Comprehensive collection of agentic patterns from Anthropic docs with Mermaid diagrams. Covers Subagent Orchestration, Progressive Skills, Parallel Tool Calling, Master-Clone Architecture, Wizard Workflows.

- [Claude Code Documentation Mirror](https://github.com/ericbuess/claude-code-docs) — Mirror of Anthropic PBC documentation, updated every few hours.

- [Claude Code Handbook](https://nikiforovall.blog/claude-code-rules/) — Best practices, tips, and techniques for Claude Code development.

- [Claude Code Infrastructure Showcase](https://github.com/diet103/claude-code-infrastructure-showcase) — Innovative approach leveraging hooks to intelligently activate appropriate Skills.

- [Claude Code PM](https://github.com/automazeio/ccpm) — Comprehensive project-management workflow with numerous specialized agents and slash-commands.

- [Claude Code Repos Index](https://github.com/danielrosehill/Claude-Code-Repos-Index) — Index of 75+ Claude Code repositories: CMS, system design, deep research, IoT, agentic workflows.

- [Claude Code System Prompts](https://github.com/Piebald-AI/claude-code-system-prompts) — All parts of Claude Code's system prompt including builtin tool descriptions, subagent prompts, utility prompts. Updated per release.

- [Claude Code Tips](https://github.com/ykdojo/claude-code-tips) — 35+ brief but information-dense Claude Code tips covering voice input, system prompt patching, container workflows.

- [Claude Code Ultimate Guide](https://github.com/FlorianBruniaux/claude-code-ultimate-guide) — Comprehensive guide from beginner to power user with production-ready templates.

- [Claude CodePro](https://github.com/maxritter/claude-codepro) — Professional development environment with spec-driven workflow, TDD enforcement, cross-session memory, semantic search.

- [ClaudoPro Directory](https://github.com/JSONbored/claudepro-directory) — Well-crafted Claude Code hooks, slash commands, subagent files.

- [Learn Claude Code](https://github.com/shareAI-lab/learn-claude-code) — Analysis of how coding agents are designed, reconstructs a rudimentary agent in Python.

- [Ralph Wiggum](https://github.com/snwfdhmp/awesome-ralph) — Ralph: AI coding technique running AI agents in automated loops until specifications are fulfilled.

- [Ralph for Claude Code](https://github.com/frankbria/ralph-claude-code) — Autonomous AI development framework enabling Claude Code to work iteratively with intelligent exit detection and safety guardrails.

- [The Ralph Playbook](https://github.com/ClaytonFarr/ralph-playbook) — Remarkably detailed guide to Ralph Wiggum technique.

**Tooling:**
Applications built on top of Claude Code with more components than slash-commands and CLAUDE.md files.

- [cc-tools](https://github.com/Veraticus/cc-tools) — High-performance Go implementation of Claude Code hooks and utilities.

- [ccexp](https://github.com/nyatinte/ccexp) — Interactive CLI tool for discovering and managing Claude Code configuration files with terminal UI.

- [cchistory](https://github.com/eckardt/cchistory) — Like shell history but for Claude Code sessions.

- [cclogviewer](https://github.com/Brads3290/cclogviewer) — Utility for viewing Claude Code .jsonl conversation files in pretty HTML UI.

- [Claude Code Templates](https://github.com/davila7/claude-code-templates) — Polished collection of resources with neatly polished UI and usage dashboard.

- [Claude Hub](https://github.com/claude-did-this/claude-hub) — Webhook service connecting Claude Code to GitHub repositories.

- [Claude Session Restore](https://github.com/ZENG3LD/claude-session-restore) — Restore context from previous Claude Code sessions by analyzing session files and git history.

- [claude-code-tools](https://github.com/pchalasani/claude-code-tools) — Well-crafted toolset for session continuity with cross-agent handoff, full-text session search, tmux-cli skill.

- [ContextKit](https://github.com/FlineDev/ContextKit) — Systematic development framework with 4-phase planning methodology and specialized quality agents.

- [recall](https://github.com/zippoxer/recall) — Full-text search Claude Code sessions from terminal.

- [Rulesync](https://github.com/dyoshikawa/rulesync) — Node.js CLI tool converting configs between Claude Code and other AI coding agents.
