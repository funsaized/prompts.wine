A curated collection of `.cursorrules` files for various technology stacks and project types. The repository aggregates community-contributed rules files that configure Cursor AI's behavior for specific frameworks, languages, and development workflows.

The collection is organized into categories including:

**Web & Mobile Frameworks:**
- Next.js, React, Vue, Svelte, React Native, Flutter
- Angular, SolidJS, Qwik, Astro
- Mobile: iOS (Swift/SwiftUI), Android (Kotlin)

**Backend & Languages:**
- Node.js, Python, Django, FastAPI, Flask
- Go, Rust, Elixir, Ruby on Rails, Laravel, Spring Boot
- .NET/C#, Java, Scala

**Specialized:**
- Testing (Playwright, Jest, Cypress, Testing Library)
- AI/ML (PyTorch, TensorFlow, Hugging Face)
- Databases (PostgreSQL, MongoDB, Redis, Supabase)
- Infrastructure (Docker, Kubernetes, Terraform)

**Key Examples:**

- [firemark/await-helper](https://github.com/firemark/await-helper) — Cursor rules for await JavaScript/TypeScript

- [cursorrules-file](https://github.com/PatrickJS/cursorrules-file) — The original .cursorrules file

- [cursorrules-file-cursor-ai-python-fastapi-api](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursorrules-file-cursor-ai-python-fastapi-api) — Python FastAPI API rules

- [typescript-nextjs-supabase-cursorrules-prompt-file](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/typescript-nextjs-supabase-cursorrules-prompt-file) — TypeScript Next.js Supabase rules

- [cursor-ai-php-laravel](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-ai-php-laravel) — PHP Laravel rules

- [cursor-ai-swift](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-ai-swift) — Swift/SwiftUI iOS development rules

- [cursor-rules-kotlin-android](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-rules-kotlin-android) — Kotlin Android rules

- [cursor-rules-rust](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-rules-rust) — Rust development rules

- [cursor-rules-go](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-rules-go) — Go development rules

- [cursor-rules-svelte](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-rules-svelte) — Svelte development rules

- [cursor-rules-flutter](https://github.com/PatrickJS/awesome-cursorrules/tree/main/rules/cursor-rules-flutter) — Flutter/Dart rules

**Contributing:** Developers can contribute by submitting pull requests with new .cursorrules files organized by technology category.
