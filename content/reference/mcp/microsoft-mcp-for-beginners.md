### Overview

This open-source curriculum introduces the fundamentals of Model Context Protocol (MCP) through real-world, cross-language examples in .NET, Java, TypeScript, JavaScript, Rust and Python. Designed for developers, it focuses on practical techniques for building modular, scalable, and secure AI workflows from session setup to service orchestration.

This curriculum is aligned with MCP Specification 2025-11-25 (the latest stable release).

### Learning Objectives

By the end of this curriculum, you will:

- Understand MCP fundamentals and why it's revolutionizing AI application integration
- Build your first MCP server in your preferred programming language
- Connect AI models to real tools and services
- Implement security best practices
- Deploy MCP solutions with confidence
- Join the MCP community

### Module Structure

#### Module 0-3: Fundamentals
- **Module 0**: Introduction to MCP - Overview of the Model Context Protocol
- **Module 1**: Core Concepts Explained - Essential building blocks of MCP
- **Module 2**: Security in MCP - Threats and best practices
- **Module 3**: Getting Started - Environment setup, basic servers/clients, integration

#### Module 3: Building Your First Server & Client
- 3.1 First Server - Create your first MCP server
- 3.2 First Client - Develop a basic MCP client
- 3.3 Client with LLM - Integrate large language models
- 3.4 VS Code Integration - Consume MCP servers in VS Code
- 3.5 stdio Server - Create servers using stdio transport
- 3.6 HTTP Streaming - Implement HTTP streaming in MCP
- 3.7 AI Toolkit - Use AI Toolkit with MCP
- 3.8 Testing - Test your MCP server implementation
- 3.9 Deployment - Deploy MCP servers to production
- 3.10 Advanced Server - Advanced features and architecture
- 3.11 Simple Auth - Auth from beginning and RBAC
- 3.12 MCP Hosts - Configure Claude Desktop, Cursor, Cline, and other MCP hosts
- 3.13 MCP Inspector - Debug and test MCP servers
- 3.14 Sampling - Collaborate with client
- 3.15 MCP Apps - Build MCP Apps

#### Module 4-5: Practical & Advanced
- **Module 4**: Practical Implementation - SDKs, debugging, testing, reusable prompt templates
- **Module 5**: Advanced Topics - Multi-modal AI, scaling, enterprise use

Advanced topics include: Azure Integration, Multi-modality, OAuth2 Demo, Root Contexts, Routing, Scaling, Security, Web Search, Realtime Streaming, Entra ID Auth, Foundry Integration, Context Engineering, Custom Transport, Protocol Features

#### Module 6-10: Community & Best Practices
- Module 6: Community Contributions
- Module 7: Lessons from Early Adoption
- Module 8: Best Practices
- Module 9: Case Studies
- Module 10: Hands-on Workshop

#### Module 11: MCP Server Database Integration
Comprehensive 13-lab hands-on learning path for PostgreSQL integration covering:
- Introduction and retail analytics use case
- Core architecture and database layers
- Security & multi-tenancy with Row Level Security
- Environment setup with Docker and Azure
- MCP Server implementation with FastMCP
- Tool development and schema introspection
- Semantic search with vector embeddings
- Testing and debugging strategies
- VS Code integration
- Docker deployment to Azure Container Apps
- Monitoring with Application Insights
- Best practices and production optimization
